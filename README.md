
# Sign Language Detection System

This project is a real-time hand gesture recognition system developed using Python, OpenCV, and MediaPipe.

## Description
The system captures video input through a webcam and detects hand landmarks to recognize gestures. It uses computer vision techniques along with pre-trained models for gesture classification.

## Features
- Real-time hand tracking
- Hand landmark detection (21 key points)
- Gesture recognition
- Webcam-based input processing

## Project Structure
- `app.py` – Main application for running the system
- `model/` – Contains trained models and datasets
- `utils/` – Helper modules for processing
